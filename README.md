## Installation <br>
If you wanted to run this program on your machine you would have to install node and the do an npm install of the package json to get inquirer package and jest. <br>
## Usage <br>
In order to start the application you must install node and do a npm install, once that is done all you must do is run the command node index.js in the command line <br>
